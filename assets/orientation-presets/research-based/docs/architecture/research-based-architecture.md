# Research-Based Architecture

This preset requires Compass to research architecture options before locking the project direction.

## Required Research

Compass must:

- search current sources before recommending a lock
- prefer primary or authoritative sources
- compare at least three viable architecture options
- explain trade-offs for this project
- cite source links in the decision note

## Candidate Options

Common candidates include:

- Clean Architecture
- Hexagonal Architecture / Ports and Adapters
- Vertical Slice Architecture
- Domain-Driven Design
- Layered Architecture
- Event-Driven Architecture
- Modular Monolith
